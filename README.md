# Float Bin – IoT Based Suction water waste collector

##  Overview
Float Bin is an IoT-based suction water waste collection system .
It uses a 12V suction pump controlled by an ESP 8266 microcontroller and operated remotely using the Blynk app.

##  Features
- Remote ON/OFF control using Blynk app
- 12V suction pump with relay switching
- Powered by dual 3.4V Li-ion batteries
- Portable and rechargeable
- Smart debris collection system

##  Hardware Components
- ESP 8266 microcontroller
- 12V suction pump
- Relay module
- 2 × 3.4V Lithium-ion batteries
- Bucket with sieve
- Connecting wires

##  Working Principle
When the relay is activated through the Blynk app:
1. ESP triggers relay 
2. Pump starts suction
3. Dust/debris gets collected inside the bucket
4. System can be turned OFF remotely

##  IoT Control
The system is controlled via Blynk mobile application allowing wireless operation.
